# Interaction Rules

## Interaction Goals
- Interactions must improve orientation, confidence, and action clarity.
- Motion and state changes must be purposeful and restrained.
- Keyboard, focus, and touch behavior must be first-class.

## Rules
- Use a single dominant interactive affordance per section.
- Ensure hover/focus states are visually distinct and accessible.
- Keep transition timing consistent (no random animation cadence).
- Avoid decorative motion that competes with copy hierarchy.
- Preserve tap-target quality on mobile; avoid dense click clusters.
- Provide clear feedback for user actions (state change, loading, result).

## CTA Interaction
- Primary CTA gets strongest contrast and spatial prominence.
- Secondary CTA should not mimic primary visual weight.
- Button labels should describe next outcome, not generic "Learn More."

## Reduced Motion
- Respect reduced motion preferences.
- Replace large movement with opacity/scale-light transitions when needed.
- Never hide critical context behind motion-only patterns.
