# Strategic Scorecard

Use this scorecard for page audits, page proposals, IA planning, and copy generation review.

## Scoring Method
- Score each dimension from 1 to 10.
- Strategic score = average of all dimensions.
- Use heuristic confidence labels: `low_heuristic`, `medium_heuristic`, `medium_high_heuristic`.

## Dimensions
1. Systems-thinking clarity
2. Revenue/commercial clarity
3. Proof integration quality
4. Differentiation vs generic agency framing
5. Fit for broken-system buyers
6. Fit for missing-system buyers
7. CTA trust-stage alignment
8. Operator/accountability clarity
9. Commercial usefulness vs generic marketing language

## Score Bands
- `>= 8.0` Strong strategic alignment (`proceed`)
- `>= 7.0 and < 8.0` Mixed strategic alignment (`proceed_with_caution`)
- `< 7.0` Strategically weak (`block_and_rework`)

## What Weak Looks Like
- Agency-style capability list with no system model.
- Size-based disqualification language.
- Only one buyer state supported.
- CTA asks for commitment before trust is earned.
- Claims without proof or implementation path.

## What Acceptable Looks Like
- Systems-first framing present but uneven.
- One buyer state is strong, second is underdeveloped.
- Basic proof/CTA coherence with notable gaps.

## What Strong Looks Like
- Clear problem -> system -> outcome narrative.
- Both broken-system and missing-system paths are supported.
- Anti-persona logic is system/mindset based, not size based.
- Proof and CTA logic align to trust stage and buyer state.

## Failure Modes to Flag
- "Enterprise-only" implication where not required.
- Generic "full-service agency" language.
- Missing-system buyers forced into high-complexity framing.
- Broken-system buyers offered only shallow tactical advice.
