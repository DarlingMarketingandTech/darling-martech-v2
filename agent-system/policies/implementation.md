# Implementation Policy

## Intent
Ship medium-scope improvements with clear boundaries and complete validation accounting.

## Required Declarations
- In-scope file groups.
- Out-of-scope file groups.
- Validation checklist tied to the active profile.

## Required Validation
- Lint and typecheck for changed code.
- Targeted checks when full build is unnecessary.
- Diff-scope evaluation against task budget.

## Completion Standard
- Final summary must include:
  - what changed
  - why it changed
  - what was intentionally not changed
  - validation evidence
