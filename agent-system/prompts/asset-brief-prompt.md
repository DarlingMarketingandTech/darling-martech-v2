# Asset Brief Prompt

Use this prompt to define an asset before generation:

1. **Business objective:** What conversion or trust problem does this asset solve?
2. **Placement:** Exact page and section where asset appears.
3. **Audience context:** Who is viewing this section and what do they need next?
4. **Asset type:** backplate, support visual, diagram, mockup composite, outcome graphic, etc.
5. **Content focus:** Key concepts or proof points to visualize.
6. **Style constraints:** premium dark-system, restrained, productized, no generic agency motifs.
7. **Dimensions/orientation:** required aspect ratio and display sizes.
8. **Interaction context:** static, subtle motion-ready, or layered background.
9. **Success criteria:** How we know the asset improves hierarchy/persuasion.
10. **Naming and mapping target:** filename format + Cloudinary folder target.

Output as a concise brief that can drive Stitch/Canva/Figma generation and downstream integration.
