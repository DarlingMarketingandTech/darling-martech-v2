# Component Visual Prompt

Use this prompt for component-level visuals:

"Create a visual asset for `{component_name}` on `{page_route}`.

Purpose:
- `{purpose}`

Placement:
- `{section_name}` using `{placement_type}` (background/support/inline).

Dimensions:
- `{ratio}` at `{primary_size}` and `{mobile_size}`.

Style:
- Premium dark-system, high clarity, restrained detail.
- Support hierarchy of `{headline_or_cta}`.
- Avoid generic agency motifs and noisy gradient clutter.

Content cues:
- `{content_notes}`

Design constraints:
- Must preserve readability for nearby text.
- Must not compete with primary CTA.
- Should reinforce proof/system framing where relevant.

Deliverables:
- 2-4 variants labeled v1-v4
- Include one recommended variant with rationale."
