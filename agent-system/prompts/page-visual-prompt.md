# Page Visual Prompt

Use this prompt for page-level visual systems:

"Create a coordinated visual set for `{page_route}` that supports narrative flow and conversion.

Required assets:
- Hero backplate
- Section support visuals for `{sections}`
- Proof support block visuals
- Optional diagram/comparison visual

Narrative goals:
- `{narrative_goals}`

Brand/system constraints:
- Premium dark-system language
- Productized strategic feel (not agency template)
- Clear hierarchy and proof-first persuasion

Technical constraints:
- Define each asset ratio, safe text area, and export dimensions.
- Ensure assets can be integrated via Cloudinary with clear naming.

Output:
1. Asset list with purpose per section
2. Prompt spec for each asset
3. Recommended style guardrails
4. Export naming strings
5. Cloudinary folder mapping"
